Antigravity Bridge

An OpenAI-compatible API bridge for your Antigravity / Gemini Code Assist account.

Un bridge con API compatible con OpenAI para tu cuenta de Antigravity / Gemini Code Assist.

What is it?

Antigravity Bridge exposes your Google-authorized Antigravity account as a standard /v1/chat/completions endpoint. This lets you connect clients like Hermes, Open WebUI, Boba, or any OpenAI-compatible tool.

Supports streaming, tools/functions, images, JSON mode, dynamic model lists, and optional API-key auth.

¿Qué es?

Antigravity Bridge expone tu cuenta de Antigravity autorizada por Google como un endpoint estándar de /v1/chat/completions. Así puedes conectar clientes como Hermes, Open WebUI, Boba o cualquier herramienta compatible con OpenAI.

Soporta streaming, tools/functions, imágenes, modo JSON, lista dinámica de modelos y autenticación opcional por API key.

Quick install

git clone https://github.com/lomeliDev/antigravity-bridge.git
cd antigravity-bridge
./install.sh

Then enable the systemd service (Linux) or run manually (macOS):

sudo cp antigravity-bridge.service.rendered /etc/systemd/system/antigravity-bridge.service
sudo systemctl daemon-reload
sudo systemctl enable --now antigravity-bridge
